1.PATH变量的认识
  1.1 你能不能在任何目录下执行某个命令，与PATH 这个变量有很大的个关系。例如当你执行ls命令时，系统就是通过path这个变量里记录的路径顺序来查找命令。
  1.2 环境变量通常是大写，为了区分自定义变量

2.基本命令  
  2.1 echo $myname #定义环境变量
  2.2 myname=zhangyuan #初始化环境变量 
  2.3 echo $HOME #显示环境变量HOME
  2.4 unset myname #取消变量
3.变量的设置规则
  3.1 等号两边的内容不能直接接空格
  3.2 变量只能是英文字母或数字，但是开头不能是数字
  3.3 双引号里的特殊字符保持特殊，单引号里特殊字符变为一般字符
  3.4 可以用\将特殊字符变为一般字符
  3.5 若一个变量要在其他子程序执行，可用export使该变量变为环境变量：export PATH

 4.3 子进程下查看父进程定义的变量
    bash #进入子进程
    echo $name  #查看name
                #显示空行
    exit        #退出子进程
    exit
    export name #将name变为环境变量
    echo $name  #查看name
    wang's name #成功显示

 4.5 单引号双引号的区别
     name=zhao     #定义并初始化变量name
     echo $name    #输出name 
     zhao
     myname=yong   #定义并初始化myname
     echo $myname  #输出
     yong
     myname="$name yong" #双引号实现 myname=name+yong
     echo $myname  #输出 
     zhao yong     #结果
     myname='$name yong' #单引号实现 
     echo $myname
     $name yong    #失败
