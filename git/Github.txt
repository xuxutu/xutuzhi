1.为什么要用github？
  借助github托管项目代码

2.基本概念
  2.1 仓库（Repository）
      仓库用来存放项目代码，一个项目对应一个仓库
  2.2 收藏（star）
      收藏项目，方便下次查看（一个项目100以上的收藏量已经很了不起了）
  2.3 fork 按钮
      完整拷贝仓库到自己的仓库
  2.4 pull request(请求将自己对fork下来的项目变动合并到原项目）
  
  2.5 watch（关注）
      watch一个项目之后，会及时查看到这个项目第一时间的更新
  2.6 issue（事务）
      发现bug后用到

 3.github注册后，创建仓库时需要邮箱验证。需要在邮箱里将github设置为白名单
  3.1 打开qq邮箱/点击设置
  3.2 点击反垃圾
  3.3 点击设置域名白名单
  3.4 输入github.com
